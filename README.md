**Modern & Responsive Task Management Web App built with MERN Stack | Secure Auth, CRUD Operations, and Vite-based Frontend**

---

## **✨ Introduction**

**Task Manager** is a **complete full-stack MERN (MongoDB, Express, React, Node.js)** application designed to help users manage daily tasks efficiently.  
It provides **authentication**, **CRUD operations**, and a **beautiful responsive interface** powered by **Vite + React**.  

This project demonstrates:

* **JWT-based Authentication** for user security  
* **RESTful API Integration** using Express  
* **MongoDB Atlas Database** for cloud data storage  
* **Component-based React architecture** with reusable UI  
* **Vercel + Render Deployment Ready**

---

## **🌟 Key Features**

* ✅ User Registration & Login 
* ✅ Create, Update, Delete Tasks  
* ✅ Responsive and Modern UI (Vite + React)  
* ✅ MongoDB Atlas integration  
* ✅ Express REST API with secure routes  
* ✅ Ready-to-deploy full-stack setup

## **🛠 Technologies Used**

* **Frontend:** React (Vite), Axios, React Router DOM  
* **Backend:** Node.js, Express.js, MongoDB  
* **Database:** MongoDB Atlas  
* **Styling:** CSS / TailwindCSS  
* **Deployment:** Vercel (Frontend), Render (Backend)

## **🚀 How to Run Locally**

1. Clone the repository:

    ```bash
   git clone https://github.com/Yashpal568/Task-Manager.git
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the development server:

   ```bash
   npm start
   ```
4. Open your browser at `http://localhost:3000`
